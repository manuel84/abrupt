# Abrupt

Tools for the AbRUPt project, See [http://wba.cs.hs-rm.de/AbRUPt/service/](http://wba.cs.hs-rm.de/AbRUPt/service/)

## Installation

Add this line to your application's Gemfile:

    gem 'abrupt'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install abrupt
